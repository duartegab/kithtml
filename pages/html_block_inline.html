Elementos de bloco e elementos inline são essenciais para o seu HTML. Enquanto os Elementos de Bloco ajudam a criar a estrutura principal da página,
os Elementos Inline dão estilo aos detalhes dentro dessa estrutura. Boa aula!

Block

  • Elementos de bloco são frequentemente usados para estruturar o layout de uma página, como seções, parágrafos, listas e divs para
    agrupamento de conteúdo. Além disso, geralmente começam em uma nova linha e ocupam toda a largura disponível do seu contêiner pai.
    Veja o exemplo do elemento de bloco <div>;

  • Coloque esses códigos no seu arquivo (o primeiro código você colocará em outro lugar. No seu explorador VSCode, adicione o arquivo
    style.css, e coloque-o lá.)

    • Arquivo HTML

    <div class="main"> <h3>Nivel de Bloco - Div</h3> <p>Este é um exemplo de uso do elemento &lt;div&gt; como um contêiner de nível de bloco.</p> </div>

    Arquivo CSS

    .main {
    background: #444040;
    }


    Ele ficará da seguinte forma em seu site:

    Título da Div

    Este é um exemplo de uso do elemento <div> como um contêiner de nível de bloco.


Inline

  • São elementos que geralmente são usados para marcar partes específicas de um texto ou conteúdo, sem quebrar a estrutura do texto.
    Eles não começam em uma nova linha e ocupam apenas o espaço necessário ao redor do conteúdo. São frequentemente usados para
    estilizar partes específicas do texto, como palavras em negrito, itálico, links, imagens inline, entre outros;

  • Coloque esses códigos no seu arquivo (o primeiro código você colocará em outro lugar. No seu explorador VSCode, adicione o arquivo
    style.css, e coloque-o lá.)

    • Arquivo CSS

    .realce {
    background-color: yellow;
    font-style: italic;
    }

    Arquivo HTML

    <p>Este é um texto com uma palavra <span class="realce">realçada</span>.</p>


    Ele ficará da seguinte forma em seu site:

    Este é um texto com uma palavra realçada.
