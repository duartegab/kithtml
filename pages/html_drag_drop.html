Esse recurso é comumente usado para facilitar a manipulação de objetos em interfaces gráficas, como arrastar arquivos para fazer upload, organizar itens em uma lista, entre outras aplicações.

Drag / Drop

  • No HTML, o drag and drop é implementado usando eventos JavaScript, como dragstart, dragover e drop, que permitem que os desenvolvedores controlem o comportamento do elemento enquanto ele está sendo arrastado e quando é solto em uma área de destino. Exemplo:

    • Em seu css:

    #drop-zone {
    width: 570px;
    height: 280px;
    border: 2px dashed #ccad9b;
    margin: 20px auto;
    text-align: center;
    line-height: 200px;
    }
    img {
    max-width: 100%;
    max-height: 100%;
    }

    Em seu html:

    <div id="drop-zone" ondrop="dropHandler(event)" ondragover="dragOverHandler(event)">
    <img src="placeholder.jpg" alt="Imagem de Exemplo"> Arraste e solte aqui </div>

    <script>
    function dragOverHandler(event) {
    event.preventDefault();
    event.dataTransfer.dropEffect = 'move';
    }

    function dropHandler(event) {
    event.preventDefault();
    const files = event.dataTransfer.files;
    for (const file of files) {
    console.log('Arquivo solto:', file.name);
    }
    }
    </script>
