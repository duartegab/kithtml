HTML semantic refere-se à prática de escrever código HTML de uma forma que forneça significado e estrutura clara ao conteúdo da página da web. Me acompanhe:

Semantics

  • O elemento <article> é uma das muitas tags HTML usada para representar um conteúdo independente e autossuficiente.
    Como um post de blog, uma notícia ou um artigo. Se liga no exemplo:

    • Arquivo html

    <article class="all-browsers">
    <h1>Sistemas Operacionais</h1>
    <article class="browser">
    <h2>Android</h2>
    <p>Desenvolvido pela Google, é um sistema operacional baseado em Linux principalmente usado em smartphones e tablets.</p>
    </article>

    <article class="browser">
    <h2>iOS</h2>
    <p>Desenvolvido pela Apple, é o sistema operacional usado em dispositivos móveis da Apple, como iPhones e iPads.</p>
    </article>

    <article class="browser">
    <h2>Linux</h2>
    <p>É um sistema operacional de código aberto que é usado em uma variedade de dispositivos, desde servidores de rede até dispositivos embarcados e computadores de mesa.</p>
    </article>
    </article>



    Arquivo css

    .all-browsers {
    margin: 0;
    padding: 5px;
    background-color: whitesmoke;
    }

    .all-browsers > h1, .browser {
    margin: 10px;
    padding: 5px;
    }

    .browser {
    background: whitesmoke;
    }

    .browser > h2, p {
    margin: 4px;
    font-size: 90%;
    }
