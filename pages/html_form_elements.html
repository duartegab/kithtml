Elementos de formulário são componentes HTML que permitem aos usuários interagir e enviar dados para um servidor da web. Eles são fundamentais para a coleta de informações dos usuários por meio de formulários em páginas da web.

input

  • O elemento <input> é usado para criar controles interativos em formulários.
    Como campos de texto, botões de rádio, caixas de seleção, etc.

    • <input type="text" name="username" placeholder="Digite seu nome de usuário">

textarea

  • O elemento <textarea> é usado para criar uma área de texto multilinha em um formulário.

    • <textarea name="message" rows="4" cols="50"> Digite sua mensagem aqui... </textarea>

select e option

  • O elemento <select> cria um menu suspenso no qual o usuário pode selecionar uma opção.
    Os elementos <option> são usados dentro do <select> para definir as opções disponíveis.

    • <select name="car">

    <option value="volvo">Volvo</option>
    <option value="saab">Saab</option>
    <option value="fiat">Fiat</option>
    <option value="audi">Audi</option>

    </select>
