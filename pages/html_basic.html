Neste primeiro capítulo, você aprenderá sobre os principais elementos que compõem o HTML. Vamos lá?!

Documentos

  • Todos os documentos HTML começam com uma declaração, informando o tipo de documento: (tag !DOCTYPE html).
    É como dizer ao construtor que tipo de casa você está construindo. Com essa declaração, o navegador sabe que está lidando com HTML.

  • O próprio documento HTML começa com a tag 'html' e termina com a tag '/html'.
    Exemplificando, é como o terreno onde a casa será construída. Todo o conteúdo da página fica dentro desse terreno.

  • A parte visível do documento HTML está entre a tag 'body' e a tag '/body'.
    Já o elemento 'body' é como o interior da casa, onde tudo o que você quer e as pessoas vejam na página fica. Textos, imagens, vídeos,
    formulários - tudo o que você vê na página fica dentro do corpo.

Vamos fazer um exemplo?


  • Dentro do seu arquivo index.html, escreva o código abaixo:

    • <!DOCTYPE html>
    <html>
    <body>

    <h1>Meu primeiro título.</h1>
    <p>Meu primeiro parágrafo.</p>

    </body>
    </html>


  • O resultado deverá aparecer da seguinte forma:

    • Meu primeiro título

    Meu primeiro parágrafo.

Links

  • Os links (que direcionam uma página a outra), são definidos pela tag:
    <a>

  • O direcionamento do link se dá pelo atributo href
    Se você ainda não entendeu esse conceito, relaxe, pois isso é assunto de um próximo capítulo!

Vamos aos exemplos?

  • Escreva o código abaixo em seu arquivo index.html:

    • <a href="https://www.google.com">Isso é um link</a>

  • Ele deverá aparecer da seguinte forma em seu site: Isso é um link


  • Ao clicar nele, você é direcionado(a) para a página do google. Que tudo, né?!


Imagens

  • As imagens do site, são definidos pela tag:
    <img>

  • Além da tag, os atributos dela também são importantes:

    src(O arquivo de origem);
    alt(Texto alternativo);
    width/height(largura e altura).

    Ficou confuso? Fique tranquilo(a)!

    Explicarei passo a passo no exemplo a seguir de como colocar uma imagem em seu site:

Se liga no exemplo

  • Baixe uma imagem em seu computador, com o formato .jpg e ronomeie ela;


  • No explorador de arquivos, tranfira a imagem para dentro do seu documento do site
    (aquele que você criou a um tempo atrás para começar os exemplos, lembra?) ;


  • Agora, passe essa imagem para dentro da pasta img, que você criou dentro de seu documento;


  • Perfeito, agora é só copiar esse código em seu VsCode
    (lembre de alterar o nome, pois minha imagem tem um nome diferente!);
