O elemento <canvas> do HTML é usado para desenhar gráficos, gráficos ou outros gráficos dinâmicos em uma página da web.
Ele fornece uma área de desenho no qual você pode usar JavaScript para desenhar gráficos em tempo real, criar animações ou interagir com os usuários.

Canvas

  • Ele é representado pelas tags e <script> . Veja só o exemplo:

    • <canvas id="myCanvas" width="200" height="100" style="border:1px solid #d3d3d3;"> Seu navegador provavelmente não suporta a tag canvas </canvas>

    <script>
    var c = document.getElementById("myCanvas");
    var ctx = c.getContext("2d");
    ctx.beginPath();
    ctx.arc(95,50,40,0,2*Math.PI);
    ctx.stroke();
    </script>


    Ele ficará da seguinte forma em seu site:

    Your browser does not support the HTML canvas tag.

    Ao usar o contexto 2D ou 3D, você pode desenhar formas, linhas, curvas, texto e imagens, manipular pixels individualmente e muito mais.
