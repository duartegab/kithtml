Em HTML, o elemento de link <a> , é uma ferramenta essencial para a navegação na web. Ele cria links entre páginas da web, recursos externos e partes específicas do próprio documento. Vamos aos exemplos:

Link HTML

  • O elemento <a> em HTML é usado para criar um link, juntamente com o atributo href é usado dentro do elemento <a> para especificar
    o destino do mesmo. Veja aqui um código simples:

    • <a href="https://www.exemplo.com">Visite o Exemplo.com</a>


    Ele ficará da seguinte forma em seu site:

    Visite o Exemplo.com


Nova guia

  • Já o atributo target controla onde o documento vinculado será aberto quando o link for clicado. O valor _blank faz com que o link seja
    aberto em uma nova guia ou janela do navegador. Código:

    • <a href="https://www.exemplo.com" target="_blank">Visite o Exemplo.com em uma nova guia</a>


    Ele ficará da seguinte forma em seu site:

    Visite o Exemplo.com em uma nova guia
