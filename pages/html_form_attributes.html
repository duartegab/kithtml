Os atributos de formulário HTML são usados para controlar o comportamento e a aparência dos elementos de formulário em uma página da web. Eles fornecem maneiras de personalizar a interação do usuário com os formulários e podem ser aplicados a diferentes tipos de elementos.

action e method

  • O atributo "action" define para onde os dados do formulário serão enviados quando submetidos, e o atributo "method" especifica o
    método HTTP a ser usado (geralmente GET ou POST).

    • <form action="/submit-form" method="post"> <!-- elementos de formulário aqui --> </form>

name

  • O atributo "name" define o nome do controle de formulário, que é usado para identificar os dados do formulário quando enviados para o servidor.

    • <input type="text" name="username" placeholder="Digite seu nome de usuário">

required

  • O atributo "required" especifica que um campo de formulário deve ser preenchido antes que o formulário possa ser enviado.

    • <input type="text" name="username" required>
