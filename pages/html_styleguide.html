Style Guide

  • Aqui vai algumas dicas para criar um bom código HTML.

  • Aqui vai algumas dicas para criar um bom código HTML.

  • Seguindo tudo o que já foi passado. É de suma importância seguir os padrões de código html.
    • Use nomes de elementos minúsculos e sempre lembrar de fechar todos eles;

      Já com os atributos, sempre use nomes minúsculos e cite os valores de cada um;

      Ah, não se esqueça de especificar altura e largura para imagens (além do texto alternativo).

    Pegou as dicas? Segue aí um exemplo:

    <h1>Tigre.</h1>
    <p>Eu sou um tigre.</p>
    <img src="/pages/img_pages/tigre.jpg" alt="Tigre" width="400" height="200">
    <a href="https://www.google.com/tigre/">Busco um tigre</a>


    Ele ficará da seguinte forma em seu site:

    Tigre.

    Eu sou um tigre.


    Busco um tigre

    Assim, identificamos o "style guide" na página. Seguindo as recomendações de escrita.
