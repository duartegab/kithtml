O elemento <audio< é usado para incorporar arquivos de áudio em um documento HTML. Ele permite que você reproduza arquivos de áudio em vários formatos,
como MP3, WAV ou OGG, diretamente na página da web.

Audio

  • Exemplo de código HTML para inserir um áudio:

    • <audio controls>

    <source src="exemplo.mp3" type="audio/mpeg">
    Seu navegador não suporta o elemento de áudio.

    </audio>


    Ele ficará da seguinte forma em seu site:

    Neste exemplo, estamos usando dois elementos <source< para fornecer opções de arquivo de áudio em diferentes formatos. O atributo controls
    adiciona controles de reprodução padrão ao elemento <audio<, permitindo que os usuários reproduzam, pausem e controlem o volume do áudio.
    Se o navegador não suportar o elemento <audio< ou nenhum dos formatos de áudio fornecidos, o texto
    "Seu navegador não suporta o elemento de áudio." será exibido.
