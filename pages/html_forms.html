Formulários HTML são elementos que permitem aos usuários inserir dados que serão enviados para um servidor para processamento adicional.
Eles geralmente contêm vários tipos de controles de entrada, como caixas de texto, botões de opção, menus suspensos, botões de envio etc.

Menu de Texto

  • O menu de texto é uma lista de opções onde o usuário pode selecionar apenas uma delas.

    Cada opção é representada por um elemento <option> dentro de um elemento <select>.

    O atributo name é usado para identificar o menu de opções quando o formulário é submetido.

    Exemplo:

    • <form action="/submit_form.php" method="post">

    <label for="name">Nome:</label>
    <input type="text" id="name" name="name">

    <label for="email">E-mail:</label>
    <input type="email" id="email" name="email">

    <input type="submit" value="Enviar">

    </form>

Menu de Seleção Múltipla

  • O menu de seleção múltipla é semelhante ao menu de texto, mas permite que o usuário selecione várias opções.

    Isso é feito definindo o atributo multiple no elemento <select>.

    Exemplo:

    • <form action="/submit_form.php" method="post">
    <label for="gender">Gênero:</label>

    <input type="radio" id="male" name="gender" value="male">
    <label for="male">Masculino</label>

    <input type="radio" id="female" name="gender" value="female">
    <label for="female">Feminino</label>

    <input type="radio" id="other" name="gender" value="other">
    <label for="other">Outro</label>

    <input type="submit" value="Enviar"> </form>

Menu de Seleção de Grupos

  • O menu de seleção de grupos é usado para organizar as opções em grupos.

    Isso é feito usando o elemento <optgroup> para cada grupo de opções dentro do elemento <select>.

    Exemplo:O menu de seleção de grupos é usado para organizar as opções em grupos.

    Isso é feito usando o elemento <optgroup> para cada grupo de opções dentro do elemento <select>.

    Exemplo:

    <option value="brasil">Brasil</option>

    <option value="uk">Reino Unido</option>
    <option value="france">França</option>
