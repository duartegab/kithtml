Um elemento HTML é uma parte fundamental na estruturação de uma página da web. Consiste em uma marca de início, algum conteúdo e uma marca de fim.
Neste segundo capítulo, iremos conhecer melhor cada um deles. Vamos lá?!

Elementos Básicos

  • Todos os elementos HTML seguem o padrão de: tag inicial, conteúdo e tag final.

  • Alguns destes elementos já foram vistos na aula anterior, pois, ao englobarem o corpo do texto, são essencias para a construção da página, como:

    • <h1> (tag inicial)		    Meu primeiro título. (conteúdo)	    <h1> (tag final)

    <p> (tag inicial)		    Meu primeiro parágrafo. (conteúdo)      <p> (tag final)

    <br> (tag inicial)                  0                                        0


  • Você percebeu que a Tag <br> não teve conteúdo nem um final?
    Essa tag pertence a um elemento vazio, vamos falar disso mais pra frente.

Elementos Aninhados

  • Os elementos HTML podem ser aninhados (ou seja, os elementos podem conter outros elementos).

  • Todos os documentos HTML se baseiam em elementos HTML aninhados.

  • No exemplo a seguir, veremos quatro elementos HTML aninhados: <html> ; <body> ; <h1> e <p> .

    • <!DOCTYPE html>
    <html>
    <body>

    <h1>Meu primeiro título.</h1>
    <p>Meu primeiro parágrafo.</p>

    </body>
    </html>


  • Dentre esses elementos aninhados, o elemento <html> é o elemento raiz, e define todo o documento;

    Assim, ele terá de se basear em outro elemento para ter mais informarções (elemento <body>);

    Que, para formar o seu corpo de texto, precisará de títulos (<h1>) e parágrafos (<p>).


  • Seguindo o exemplo, o seu texto ficará da seguinte forma:

    • Meu primeiro título.

    Meu primeiro parágrafo.

Elementos Vazios

  • Elementos HTML sem conteúdo são elementos vazios.

  • A tag inicial gera uma quebra de linha e é um elemento vazio, sem conteúdo e uma tag de fechamento: <br>

    • Vamos aos exemplos?

  • Escreva o código abaixo em seu index.html para fazer um exemplo:

    • <p>Isso é um <br> parágrafo com uma quebra de linha.<p>

  • Seguindo o material, o texto aparecerá no site da seguinte forma:

    Isso é um
    parágrafo com uma quebra de linha
