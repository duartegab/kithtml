O atributo "id" em HTML é uma forma de identificar exclusivamente um elemento na página. Assim como o atributo "class", podemos estilizá-lo com diversas coisas.

Id

  • O valor do atributo "id" deve ser único em todo o documento HTML e deve começar com uma letra (A-Z ou a-z) seguida de letras, dígitos (0-9), hífens ou sublinhados. Vamos fazer um exemplo?

    • No seu arquivo html:

    <h1 id="annunc">Anuncie aqui: Anúncie</h1>


    No seu arquivo css:

    #annunc {
    background-color: lightblue;
    color: black;
    padding: 40px;
    text-align: center;
    width: 800px;
    }


    Ele ficará da seguinte forma em seu site:

    Anuncie aqui:

    Assim, identificamos o <h1> na página. Com cor de funfo, cor de texto, margem e espaçamento.
