Os elementos de comentários no HTML são usados para inserir notas ou explicações diretamente no código HTML, sem que essas informações sejam exibidas na página da web.
Eles são úteis para os desenvolvedores adicionarem documentação, anotações ou observações sobre o código, ajudando na compreensão e manutenção do mesmo. Vamos lá?!

Comments HTML

  • A tag <!-- --> é usada para adicionar comentários no código HTML, fornecendo informações para os desenvolvedores que não são exibidas no navegador.
    Exemplo de código:

    • <!-- Este é um comentário HTML -->
    <p>Este texto será exibido na página.</p>


    Ele ficará da seguinte forma em seu site:

    Este texto será exibido na página.
