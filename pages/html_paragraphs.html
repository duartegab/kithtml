Parágrafos HTML são os conjuntos de ideia, blocos de textos. Parágrafos é tudo aquilo que dá sentido para seus títulos.

Parágrafos HTML

  • A tag <p> define um parágrafo.

    Um parágrafo começa sempre em uma nova linha, e os usuários costumam colocar uma margem, antes e depois dele.


    Se liga nos exemplos:

    <p>Isso é um parágrafo.</p>
    <p>Isso é um outro parágrafo.</p>


  • O resultado deverá aparecer da seguinte forma:

    • Isso é um parágrafo.

    Isso é outro parágrafo.

Elemento br

  • O elemento vazio <br> gera uma quebra de linha. Use-o quando quiser pular uma linha sem a necessidade de criar outro parágrafo.

  • Escreva o código abaixo em seu index.html para fazer um exemplo:

    • <p>Isso é um <br> parágrafo com uma quebra de linha.<p>

  • Seguindo o material, o texto aparecerá no site da seguinte forma:

    Isso é um
    parágrafo com uma quebra de linha

    • A tag <hr> é uma tag vazia, então ela não tem uma tag final.
